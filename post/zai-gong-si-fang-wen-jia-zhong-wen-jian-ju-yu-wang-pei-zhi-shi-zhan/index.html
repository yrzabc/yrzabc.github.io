在公司访问家中文件 - 局域网配置实战

2022-04-14

通过这篇文章，你将可以实现以下目标:

  • 异地访问任意设备
    • 需提前进行配置并确保该设备联网
  • 不同设备聊天，文件传输
    • 相较于微信，具有不限制文件大小和到期时间，采用局域网传输，安全保密的优势
  • 网络共享(可以共享校园网, 不是开热点共享, 可以远程共享, 见附件)

准备工作

1.前往Zerotier官网下载并安装软件，所有计划连接的设备都需要安装;
2.在飞鸽传书官网下载并安装软件, 有需要进行聊天和文件传输的设备安装该软件;
 

局域网连接

1.准备本地电脑访问

MAC

1.打开mac自带的系统终端；

2.

如果你的 Python 版本是 3.X
python -m http.server <your-port>
如果你的 Python 版本是 2.X
python -m SimpleHTTPServer <your-port>
如果你不清楚，可以两段代码都实验一下

Windows1

1.下载工具；
Everything: 可以快速定位文件和文件夹，适合只需要进行访问的情形


2.配置工具，搭建共享网站；
  • 打开everything软件；
  • 按下Ctrl + P快捷键调出设置页面，选择http服务器，勾选开启http服务器，设置访问端口，用户名和密码，勾选允许http下载，确认；

    • Windows2

    1.下载工具；
    webd: 轻量局域网共享软件，适合需要对文件进行上传，删除的情形


    2.配置工具，搭建共享网站；
    • 打开webd配置文件( webd.conf )；
      • 修改共享文件夹位置；
        # for windows
        Webd.Root "你自己想分享的文件路径"
        # 注意把.Trash移动到共享文件夹目录中，否则后续无法删除文件
      • 修改访问权限
        # user1 has all permissions.
        Webd.User rlumS name password
      • 可以根据官网文档自己设置其他访问用户，访问端口默认9212，也可以参考修改

    3.双击tools.cmd运行

    Tips:
    配置完成后可以在本地浏览器上输入http://您电脑的IP地址:端口号，倘若可以访问成功则配置完成；
     

    搭建虚拟局域网(100Mbps)

    1.前往Zerotier官网注册账号；
    2.点击网站Network选项，选择创建新网络；

    3.进入你创建的网络；

  • 在Basics - Access Contro中将访问权限改为public,
  • 在Advanced - IPv4 Auto-Assign 中选择一个你喜欢的网段，
  • 记下Basics - Network ID中的代码，这是你局域网的名字
    •  

    4.在所有你想连接的设备上进入Zerotier，输入你的Network ID，点击加入；
     
    5.在网页端刷新，可以在members中看到你加入的设备，可以自定义备注，点击设置，勾选允许桥接;

  • 可以在客户端查看到各个客户端的ip地址，对应网页就可以确认具体是哪一台设备

    • 愉快跨设备访问

       倘若你完成了上述所有的步骤，那么此时你在任意一台设备的浏览器中输入以下网址
    http://客户端ip地址:客户端端口号，你就能异地跨设备访问啦！

    Tips
    客户端ip地址就是你在某台设备上开启Zerotier后自动分配的ip地址

    端口号是你第一步配置本地电脑时设置的端口，mac上默认8000，everything3000，webd默认9212

    原理: 利用Zerotier搭建一个虚拟局域网，这样即使设备处于不同网络也可以进行局域网内互相访问的操作，免费版Zerotier提供100Mbps带宽，最多50台设备的局域网~~
